## Install via pip: 
```pip install argai```

## Setup your LLM:

### OpenAI
Can be found at https://platform.openai.com/account/api-keys.

via terminal:
```
export OPENAI_API_KEY="..."
```

via python:
```python
import openai
openai.api_key = "..."
```
