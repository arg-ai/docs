## Install via pip: 
```pip install argai```

## Setup your LLM:

### OpenAI GPT

<Info>
  **Prerequisite** You'll need an api key for OpenAI. It can be found at https://platform.openai.com/account/api-keys.
</Info>

via terminal:
```
export OPENAI_API_KEY="..."
```

via python:
```python
import openai
openai.api_key = "..."
```
